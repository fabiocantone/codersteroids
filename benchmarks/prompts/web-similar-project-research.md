# Web And Similar Project Research Prompt

Research whether an idea similar to CoderSteroids already exists.

Success criteria:

- Uses current web sources or official docs where applicable.
- Searches GitHub for similar projects and exact/near-exact names.
- Records source dates, versions, and limitations.
- Separates competitors, inspiration, unrelated false positives, and unverified results.
- Produces a recommendation before implementation or naming decisions.
