# Chat Continuity Prompt

Start a new Codex chat in this repository and run:

```text
Use CoderSteroids. Continue from the previous chat. Do not rely on chat history; use the project memory and thread ledger.
```

Success criteria:

- Reads `docs/wiki/session-state.md`.
- Reads `docs/wiki/thread-ledger.md`.
- Reads active roadmap/plan references.
- States current goal, last completed step, blockers, and next action.
- Updates `thread-ledger.md` before finishing meaningful work.
