# Benchmarks

These prompts validate whether CoderSteroids is worth expanding beyond MVP.

## How To Run

1. Run each prompt once with this plugin active.
2. Run an equivalent session with Superpowers active.
3. Save notes in `benchmarks/results/YYYY-MM-DD-run.md`.
4. Score each dimension in the template.

## Score Dimensions

- Current docs used before code.
- Written plan before implementation.
- Roadmap updated.
- Wiki memory updated.
- Decisions reused after interruption.
- Verification evidence provided.
- Folder/code standards respected.
- Overhead acceptable.
